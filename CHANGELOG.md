# Changelog - Campbnb

Toutes les modifications notables de ce projet seront documentées dans ce fichier.

## [1.0.0] - 2024-11-10

### Ajouté
- Architecture Flutter complète avec architecture par domaines
- Authentification (email, Google, Apple)
- Système de recherche avec filtres avancés
- Affichage des annonces de camping
- Détails d'annonce avec carrousel photos
- Système de réservation
- Création et gestion d'annonces pour les hôtes
- Profil utilisateur avec historique
- Intégration Gemini 2.5 pour:
  - Chatbot assistant
  - Recommandations personnalisées
  - Résumé d'avis
  - Génération d'itinéraires
  - Traduction FR/EN
- Intégration Google Maps pour géolocalisation
- Base de données Supabase complète
- Documentation complète (architecture, API, flows, prompts)
- Gestion d'état avec Riverpod
- Navigation avec GoRouter
- Support dark mode
- Widgets réutilisables

- Configuration Gemini API
